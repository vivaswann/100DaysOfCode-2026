## 1. Palindrome Linked List (LeetCode #234)
A problem that teaches linked list traversal and two-pointer techniques by checking whether a linked list reads the same forward and backward. Given the head of a singly linked list, return true if the linked list is a palindrome, otherwise return false.

This problem is commonly asked in interviews and helps build concepts like:
- Linked list traversal
- List reversal

These are important for efficient linked-list problems.

**Your task:** Determine whether the linked list is a palindrome using an optimized approach.

### Examples

**Input:**
```
head = [1,2,2,1]
```
**Output:**
```
true
```

---

**Input:**
```
head = [1,2]
```
**Output:**
```
false
```

---

## 2. Detect Cycle in a Linked List
A problem that teaches linked list traversal and cycle detection techniques by checking whether a linked list contains a loop. Given the head of a singly linked list, determine whether the linked list contains a cycle. A cycle exists if a node can be reached again by continuously following the next pointers.

This problem is commonly asked in interviews and helps build concepts like:
- Linked list traversal
- Cycle detection

These are important for efficient linked-list operations.

**Your task:** Determine whether the linked list contains a cycle using an optimized approach.

### Examples

**Input:**
```
head = [3,2,0,-4], pos = 1
```
**Output:**
```
true
```

---

**Input:**
```
head = [1,2], pos = 0
```
**Output:**
```
true
```

---

**Input:**
```
head = [1], pos = -1
```
**Output:**
```
false
```
