## 1. Assign Cookies (LeetCode #455)
A problem that teaches two pointer techniques by assigning cookies to children based on their greed factor. Given two integer arrays: g[] representing the greed factor of children, and s[] representing the size of cookies. A child is satisfied if the cookie size is greater than or equal to the child's greed factor. Return the maximum number of children that can be satisfied.

This problem is commonly asked in interviews and helps build concepts like:
- Two pointers
- Greedy algorithms

which are important for optimization problems.

**Your task:** Find the maximum number of satisfied children using an optimized approach.

### Examples

**Input:**
```
g = [1,2,3], s = [1,1]
```
**Output:**
```
1
```

---

**Input:**
```
g = [1,2], s = [1,2,3]
```
**Output:**
```
2
```

## 2. Backspace Equality
A problem that teaches string traversal techniques by comparing two strings after applying backspace operations. Given two strings s and t containing lowercase letters and the '#' character, process both strings where '#' represents a backspace operation. Return true if both processed strings are equal, otherwise return false.

This problem is commonly asked in interviews and helps build concepts like:
- Two pointers
- String processing

which are important for efficient text-manipulation problems.

**Your task:** Determine whether both processed strings are equal using an optimized approach.

### Examples

**Input:**
```
s = "ab#c", t = "ad#c"
```
**Output:**
```
true
```

---

**Input:**
```
s = "ab##", t = "c#d#"
```
**Output:**
```
true
```

---

**Input:**
```
s = "a#c", t = "b"
```
**Output:**
```
false
```
