## 1. Move Zeroes (LeetCode #283)
A problem that teaches two pointer techniques by moving all zeroes in an array to the end while maintaining the relative order of the non-zero elements. Given an integer array nums, move all 0's to the end of it while maintaining the relative order of the non-zero elements.

**Note:** You must do this in-place without making a copy of the array.

This problem is commonly asked in interviews and helps build concepts like:
- Two pointers
- In-place array manipulation

which are important for efficient array problems.

**Your task:** Move all zeroes in the array to the end using an optimized approach.

### Examples

**Input:**
```
nums = [0,1,0,3,12]
```
**Output:**
```
[1,3,12,0,0]
```

---

**Input:**
```
nums = [0]
```
**Output:**
```
[0]
```

---

**Input:**
```
nums = [1,2,3]
```
**Output:**
```
[1,2,3]
```

## 2. Card Picking Game
A problem that teaches two pointer techniques by simulating a two player card picking game. Given an array of card values, two players alternately pick a card from either the left end or the right end of the array, always picking whichever end has the larger value. Player 1 picks first, followed by Player 2, and so on, until all cards are picked.

This problem is commonly asked in interviews and helps build concepts like:
- Two pointers
- Simulation
- Greedy selection

which are important for efficient array traversal problems.

**Your task:** Determine the final scores of both players using an optimized approach.

### Examples

**Input:**
```
cards = [5,3,7,2]
```
**Output:**
```
Player 1 = 12, Player 2 = 5
```

---

**Input:**
```
cards = [8,4,1,9,6]
```
**Output:**
```
Player 1 = 18, Player 2 = 10
```
