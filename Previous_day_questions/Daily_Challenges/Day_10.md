## 1. Find the Index of the First Occurrence in a String (LeetCode #28)
A problem that focuses on basic string searching by finding the first occurrence of one string inside another. Given two strings haystack and needle, return the index of the first occurrence of needle in haystack. If needle is not present, return -1.

This problem helps build concepts like:
- String traversal
- Substring searching
- Character comparison
- Basic string manipulation

which are important for solving a variety of string-searching problems.

**Your task:** Find the first occurrence of the given substring using an optimized approach.

### Examples

**Input:**
```
haystack = "sadbutsad", needle = "sad"
```
**Output:**
```
0
```

---

**Input:**
```
haystack = "leetcode", needle = "leeto"
```
**Output:**
```
-1
```

---

**Input:**
```
haystack = "hello", needle = "ll"
```
**Output:**
```
2
```

## 2. Determine Gender
A problem that focuses on analyzing the unique characters present in a username. Based on the number of distinct characters, determine the appropriate message to display.

Given a username consisting of lowercase English letters:
- If the number of distinct characters is even, print "CHAT WITH HER!"
- If the number of distinct characters is odd, print "IGNORE HIM!"

This problem helps build concepts like:
- String traversal
- Character comparison
- Distinct character identification
- Basic string manipulation

which are important for solving a variety of string-based problems.

**Your task:** Determine the correct output based on the number of distinct characters in the username.

### Examples

**Input:**
```
username = "alex"
```
**Output:**
```
CHAT WITH HER!
```

---

**Input:**
```
username = "abc"
```
**Output:**
```
IGNORE HIM!
```

---

**Input:**
```
username = "sevenkplus"
```
**Output:**
```
CHAT WITH HER!
```
