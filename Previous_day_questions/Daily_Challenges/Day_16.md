## 1. Swap Nodes In Pairs (LeetCode #24)
A problem that teaches linked list traversal and pointer manipulation techniques by swapping adjacent nodes in pairs. Given the head of a linked list, swap every two adjacent nodes and return the head of the modified list.

This problem is commonly asked in interviews and helps build concepts like:
- Linked list traversal
- Pointer manipulation
- Node swapping

These are important for efficient linked-list operations.

**Your task:** Swap every two adjacent nodes in the linked list using an optimized approach.

### Examples

**Input:**
```
head = [1,2,3,4]
```
**Output:**
```
[2,1,4,3]
```

---

**Input:**
```
head = []
```
**Output:**
```
[]
```

---

**Input:**
```
head = [1]
```
**Output:**
```
[1]
```

---

## 2. Delete Last Node of a Doubly Linked List
A problem that teaches doubly linked list traversal and pointer manipulation techniques by deleting the last node of a doubly linked list. Given the head of a doubly linked list, delete the last node and return the updated head of the linked list.

This problem helps build concepts like:
- Doubly linked list traversal
- Pointer manipulation
- Node deletion

These are important for efficient linked-list operations.

**Your task:** Delete the last node of the doubly linked list using an optimized approach.

### Examples

**Input:**
```
1 <-> 2 <-> 3 <-> 4
```
**Output:**
```
1 <-> 2 <-> 3
```

---

**Input:**
```
10 <-> 20
```
**Output:**
```
10
```

---

**Input:**
```
5
```
**Output:**
```
NULL
```
